[算法随笔]std::unordered_set用法

二月 14, 2026 阅读时间：1 分钟

unordered_set 不保证元素有序性，能在 $O(1)$ 的时间内完成查找，增加和删除任务

构造函数 链接到标题

创建一个空的 unordered_set

std::unordered_set<int> uset;

插入元素 链接到标题

使用 insert() 方法

uset.insert(10)

查找元素 链接到标题

使用 find() 方法

auto it = uset.find(10);
if(it != uset.end()) {
	// 元素存在
}

使用 contains() 方法

if(uset.contains(10)) {
	// 元素存在
}

删除元素 链接到标题

删除指定元素 链接到标题 

uset.erase(10);

利用迭代器删除 链接到标题 

auto it = uset.find(10);
uset.erase(it);

删除范围内元素 链接到标题 

uset.erase(uset.begin(), uset.end());

删除所有元素 链接到标题 

uset.clear();

获取大小和空检查 链接到标题 

size_t size = uset.size();
bool isEmpty = uset.empty();
